## 1.Null safety

코틀린은 'non-nullable'을 기본으로 해 null 참조를 제거한다. 컴파일러가 초기화되지 않는 사용을 막는다. null 값을 가지고 싶으면 그 타입을 'nullable'로 선언해야 한다.

```markdown
# nullable
var nullable: String? = null

# non-nullable
if (nullable != null) nullable.doSomething()
nullable?.doSomething() //safe call
```
